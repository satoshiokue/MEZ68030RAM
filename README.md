# MEZ68030RAM
68030 Mezzanine board for EMUZ80

MC68030の配線が必要なピンは下側にアンダーラインがあります。  
汎用ICソケットを使用ピンとMPUを取り外しやすいように四隅に配置します。  

![MEZ68030 PCB TOP](https://github.com/satoshiokue/MEZ68030/blob/main/MEZ68030_socket.jpg)
 
10k x2  
0.1uF x3  
47uF/25V x1
